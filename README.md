# Diff my image - Web

Sitio web oficial para la aplicación **Diff my image**, un comparador de imágenes con algoritmos SSIM y MSE.

## 🚀 Tecnologías

Este sitio está desarrollado con:
- [Astro](https://astro.build/)
- Bootstrap 5
- KaTeX (para fórmulas matemáticas)

## 📄 Licencia

Este proyecto está bajo la licencia **GPLv3**.

## 🧞 Comandos

Todos los comandos se ejecutan desde la raíz del proyecto, en la terminal:

| Comando | Acción |
| :--- | :--- |
| `npm install` | Instala dependencias |
| `npm run dev` | Inicia el servidor de desarrollo en `localhost:4321` |
| `npm run build` | Construye el sitio para producción en `./dist/` |
| `npm run preview` | Previsualiza la construcción localmente |
